# Food Traceability System using Blockchain

Hệ thống truy xuất nguồn gốc thực phẩm sử dụng công nghệ blockchain

## Công nghệ sử dụng
- Frontend: React.js + TypeScript + Material-UI
- Backend: Node.js + Express.js
- Blockchain: Ethereum/Smart Contract
- Database: MongoDB
- Authentication: JWT

## Các tính năng chính
- Đăng ký/Đăng nhập cho người dùng và nhà cung ứng
- Quản lý thông tin sản phẩm
- Theo dõi chuỗi cung ứng
- Mua bán sản phẩm
- Xác thực nguồn gốc thực phẩm

## Cài đặt và chạy
1. Clone repository
2. Cài đặt dependencies:
   ```bash
   cd frontend
   npm install
   cd ../backend
   npm install
   ```
3. Chạy frontend:
   ```bash
   cd frontend
   npm start
   ```
4. Chạy backend:
   ```bash
   cd backend
   npm start
   ```
